# 基本試験計画書 - Project Back-Scheduler

## 1. 概要

### 1.1 目的
本ドキュメントは、Project Back-Schedulerの基本機能が正しく動作することを確認するための試験計画を定義する。

### 1.2 対象バージョン
- v1.2.0以降

### 1.3 試験環境
- ブラウザ: Chrome (最新版)
- OS: macOS / Windows 10以降
- 開発サーバー: `python3 -m http.server 8080` または同等

---
